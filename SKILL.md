# A股盘前短线标的筛选 v6.9.36

## 版本历史
- **v6.9.36**: 行业补全仅周日全量拉取，其他日仅读取缓存；二级行业替换主营业务列；_load_industry_cache兼容旧dict格式
- **v6.9.35**: 二级行业缓存（东方财富sshy）；步骤10H从CompanySurvey读取sshy替代CoreConception主营业务
- **v6.9.34**: 东方财富HTTP行业分类替代Baostock TCP；新增证监会→申万映射表

### 步骤10A: 全市场拉取

### 步骤10B: 行业补全
**v6.9.36**: 仅周日执行全量HTTP拉取更新缓存，其他日仅读取缓存。
- 周日: 东方财富HTTP API拉取所有缺失的行业分类（一级+二级）
- 其他日: 仅从 `行业缓存.json` 和 `二级行业缓存.json` 读取
- 行业查表优先级: 硬编码覆盖 → 东方财富HTTP缓存 → 代码段映射

## 周日自动化
使用 `sunday_industry_pull.py` 脚本在每周日执行行业补全全量拉取：
- 拉取全量A股行业分类（一级+二级）
- 更新 `行业缓存.json` 和 `二级行业缓存.json`
- 自动推送到GitHub仓库
